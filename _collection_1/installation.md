### Install using pip
```
pip install nn4n
```

### Install from GitHub
#### Clone the repository
```
git clone https://github.com/zhaozewang/NN4Neurosci.git
```
#### Navigate to the NN4Neurosci directory
```
cd NN4Neurosci/
pip install -e .
```
