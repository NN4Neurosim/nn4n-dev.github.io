# Install using pip
```
pip install nn4n
```

# Install from Source
#### Clone the repository
```
git clone https://github.com/NN4Neurosim/nn4n.git
```
#### Navigate to the NN4Neurosci directory
```
cd nn4n/
pip install -e .
```
